    Quem somos?

    Pythonicas é um grupo de estudo participativo criado para mulheres aprenderem a programar na linguagem Python.

    Para mulheres?

    Sim, essa é uma maneira de empoderar mulheres e contribuir para que outras desperte interesse para ciência, tecnologia e engenharia. Ou até mesmo mulheres se divertir programando ou aplicar programação no que quiser e imaginar.

    Por que Python?

    Python é uma linguagem de programação fácil de aprender e que pode ser aplicada em diversas áreas. Por exemplo: Desenvolvimento web, robótica, astronomia, ciência de dados, computação científica e matemática, games, jornalismo de dados, redes de computadores, animação... Quase tudo que você quiser. Além disso, python favorece focarmos no que mais interessa: descobrir como solucionar problemas sem perder tempo com detalhes desnecessários na escrita/sintaxe. Afinal, Python é uma linguagem elegante e intuitiva, amiga das/os programadoras/es.

    Quem pode participar?

    Não importa seu nível de conhecimento, sua "área"(exatas, humanas, saúde) ou sua profissão. Você é bem-vinda e o único pré-requisito: Vontade aprender. As demais habilidades que "faltar", técnicas e/ou humanas, vamos desenvolver juntas!
